## 🚀 GETTING STARTED

### 📋 Prerequisites
```text
Node.js 18+     ✅ Required
npm 9+          ✅ Required
Git             ✅ Required
Google Account  ✅ For deployment only
```

### ⚡ Quick Start
```bash
# 📡 STEP 1: Clone the repository
git clone https://github.com/SandipGhorai-max/CARBONTRACE.git

# 📁 STEP 2: Enter the command center
cd CARBONTRACE

# 📦 STEP 3: Install all systems
npm install

# 🔧 STEP 4: Configure environment
cp .env.example .env
# → Add your Firebase credentials to .env

# 🚀 STEP 5: Launch development server
npm run dev

# 🌐 STEP 6: Open in browser
# → http://localhost:5173
```

### 🧪 Run Tests
```bash
# Run all tests
npm test

# Run with coverage report
npm run test:coverage

# Run linter
npm run lint
```

### 🐳 Deploy to Google Cloud
```bash
# One command deployment
chmod +x deploy.sh && ./deploy.sh

# Manual deployment
gcloud auth login
gcloud config set project virtual-promptwar-492913
gcloud builds submit --tag gcr.io/virtual-promptwar-492913/carbontrace .
gcloud run deploy carbontrace \
  --image gcr.io/virtual-promptwar-492913/carbontrace \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated
```

## 🌌 About The Project

**CarbonTrace** is a **next-generation AI Climate Command Center** that transforms personal carbon footprint tracking into an immersive, futuristic experience. Built for the **VirtualPrompt War** competition, it combines real-time emissions intelligence with a stunning deep-space aesthetic to make climate action feel like piloting a spacecraft. Every interaction — from logging a car trip to deploying an optimization protocol — is wrapped in animated neon visuals, procedural sound design, and AI-driven insights.

> 🤖 *"The robot isn't just decoration — it's your AI co-pilot for the planet."*

At the heart of CarbonTrace lives an **animated AI Robot Assistant** that breathes, scans, and thinks in real time. Its eyes shift from **🟢 green** (planet safe) to **🟡 amber** (caution) to **🔴 red** (critical) based on your carbon footprint. A chest-mounted display shows your live CO₂ reading, while a typewriter speech bubble delivers context-aware intelligence: *"SYSTEMS NOMINAL. PLANET TRAJECTORY: STABLE."* When you submit data, the robot enters a processing state — eyes spin, chest flickers — before returning its verdict.

The technical achievement spans **13+ custom React components**, **78 passing tests** across 5 suites, **Firebase real-time sync**, **Google Cloud Run** deployment with Docker multi-stage builds, full **WCAG 2.1 AA accessibility**, and a **Web Audio API** sound engine generating procedural audio without any external files. Every CO₂ factor is sourced from the **EPA** and **Our World in Data**, with a smart 7-day averaging system that prevents statistical extrapolation from destroying accuracy. This isn't a prototype — it's a production-grade climate intelligence platform. ⚡

## 🚀 Getting Started

### Prerequisites

- **Node.js** 18+ and **npm** 9+
- **Google Cloud SDK** (for deployment)
- A Firebase project (optional — runs offline without it)

### Installation

```bash
# Clone the repository
git clone https://github.com/SandipGhorai-max/CARBONTRACE.git

# Navigate to project
cd CARBONTRACE

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```bash
# Build optimized bundle
npm run build

# Preview production build
npm run preview
```

## 🔒 Security

| Measure | Details |
|:---|:---|
| 🧹 **Input Sanitization** | All form fields validated before processing |

## 📦 Deployment

### One-Command Deploy

```bash
# Authenticate with Google Cloud
gcloud auth login

# Set project
gcloud config set project virtual-promptwar-492913

# Enable required services
gcloud services enable run.googleapis.com containerregistry.googleapis.com

# Build and submit to Cloud Build
gcloud builds submit --tag gcr.io/virtual-promptwar-492913/carbon-trace .

# Deploy to Cloud Run
gcloud run deploy carbon-trace \
  --image gcr.io/virtual-promptwar-492913/carbon-trace \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --port 8080
```

### Docker Local Build

```bash
# Build Docker image locally
docker build -t carbon-trace .

# Run container
docker run -p 8080:8080 carbon-trace
```

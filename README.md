## 🤖 About CarbonTrace

> *"The most sophisticated personal carbon footprint tracker ever built for a hackathon — combining AI robotics, real-time Firebase sync, and Google Cloud infrastructure into a single jaw-dropping experience."*

**CarbonTrace** is not just an app. It is a **mission control center for the planet**. Built for the **VirtualPrompt War** competition, it transforms the mundane task of carbon tracking into an immersive, futuristic experience powered by AI, real-time data, and stunning visuals. Every interaction — from logging a car trip to deploying an optimization protocol — is wrapped in animated neon visuals, procedural sound design, and AI-driven insights.

### 🌍 Why It Exists

Every year, the average human produces **4 tons of CO₂** — but most people have no idea where it comes from or how to reduce it. CarbonTrace changes that by making climate data **personal, visual, and actionable**. All CO₂ factors are sourced from the **EPA** and **Our World in Data** with a smart 7-day averaging system that prevents statistical extrapolation from destroying accuracy.

### 🤖 The AI Robot

At the heart of CarbonTrace is an **animated AI robot assistant** that **watches your data in real time**, changes its emotional state based on your footprint, and delivers personalized insights through a typewriter speech bubble — like having a climate scientist living inside your browser.

- **🟢 Green Eyes** — *"SYSTEMS NOMINAL. PLANET TRAJECTORY: STABLE."*
- **🟡 Amber Eyes** — *"WARNING: ELEVATED EMISSIONS DETECTED. OPTIMIZING..."*
- **🔴 Red Eyes** — *"CRITICAL ALERT: CARBON THRESHOLD EXCEEDED. INITIATING PROTOCOLS."*

The robot's head scans left and right, arms breathe up and down, antenna pulses signal rings, and a chest-mounted display shows your live CO₂ reading. When you submit data, the robot enters a **processing state** — eyes spin, chest flickers — before delivering its verdict.

### ⚡ The Technology

Built on **React 18**, deployed on **Google Cloud Run**, backed by **Firebase Realtime Database**, containerized with **Docker**, and tested with **78 passing tests** — CarbonTrace represents a complete, production-grade full-stack application built in a competitive hackathon environment. The **Web Audio API** generates procedural sounds without any external files. The entire app works offline with localStorage fallback.

## 🚀 Getting Started

### Prerequisites

- **Node.js** 18+ and **npm** 9+
- **Google Cloud SDK** (for deployment)
- A Firebase project (optional — app works fully offline)

### Installation

```bash
# 🔽 Clone the repository
git clone https://github.com/SandipGhorai-max/CARBONTRACE.git

# 📁 Navigate to project
cd CARBONTRACE

# 📦 Install dependencies
npm install

# 🔧 Set up environment variables (optional — for Firebase)
cp .env.example .env
# Add your Firebase config to .env

# 🚀 Start development server
npm run dev

# 🧪 Run tests
npm test

# 📊 Run tests with coverage report
npm run test:coverage

# 🏗️ Build for production
npm run build

# 👁️ Preview production build
npm run preview
```

> 💡 **Tip:** The app works fully without Firebase — data persists in localStorage. Add Firebase credentials to `.env` for cloud sync.

## 🐳 Deployment

### One-Command Deploy

```bash
chmod +x deploy.sh && ./deploy.sh
```

### Manual Steps

```bash
# 🔐 Authenticate with Google Cloud
gcloud auth login

# 📋 Set project
gcloud config set project virtual-promptwar-492913

# ☁️ Enable required services
gcloud services enable run.googleapis.com containerregistry.googleapis.com

# 🏗️ Build and push Docker image via Cloud Build
gcloud builds submit --tag gcr.io/virtual-promptwar-492913/carbon-trace .

# 🚀 Deploy to Cloud Run
gcloud run deploy carbon-trace \
  --image gcr.io/virtual-promptwar-492913/carbon-trace \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --port 8080
```

### Live URL

```
🌐 https://carbon-trace-78346056981.us-central1.run.app
```

### Docker Local Build

```bash
# Build image locally
docker build -t carbon-trace .

# Run container
docker run -p 8080:8080 carbon-trace

# Visit http://localhost:8080
```
